공공데이터 수집, 분석·시각화, 프로젝트 고도화, 발표 문서화까지 연결합니다.

전체 과정

대상 학생

  • 기초 문법과 응용 문제 해결 경험이 있는 학생
  • AI·데이터 분석 프로젝트를 직접 만들어 보고 싶은 학생
  • 공공데이터를 활용한 탐구 보고서를 준비하는 학생
  • 발표 자료와 포트폴리오 결과물이 필요한 학생

학습 목표

  • 문제 정의부터 데이터 수집까지 프로젝트 흐름 이해
  • 데이터 전처리와 시각화를 통한 분석 능력 강화
  • AI 도구를 활용한 탐구 질문 확장과 해석 보완
  • 보고서·발표자료·포트폴리오 결과물 완성

구성 요소

1

데이터 수집·전처리

공공데이터를 찾고 필요한 컬럼을 정리하며 분석 가능한 형태로 데이터를 가공합니다.

2

분석·시각화

Python을 활용해 데이터를 비교·분석하고 그래프와 표로 결과를 시각화합니다.

3

프로젝트 고도화

분석 결과를 바탕으로 인사이트를 정리하고 AI 도구를 활용해 해석을 보완합니다.

4

발표·포트폴리오

결과물을 보고서, 발표자료, 포트폴리오 형태로 정리하여 제출 가능한 산출물로 완성합니다.

클래스 연결

권장 운영: 주 1회 8주. 결과물은 분석 코드, 시각화 이미지, 발표자료, 포트폴리오 페이지로 축적합니다.
