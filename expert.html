Expert 과정

데이터 프로젝트와 포트폴리오로 진로 확장까지 연결하는 과정

심화 학습을 원하는 학생을 대상으로 실제 데이터를 분석하고 시각화하며, 발표 가능한 프로젝트 결과물과 포트폴리오를 완성합니다.

클래스 구성 보기전체 과정

대상 학생

  • AI·데이터 분석·컴공 계열 진로를 고려하는 학생
  • 대회, 보고서, 포트폴리오 결과물이 필요한 학생
  • 기초 문법을 넘어 실제 문제를 설계하고 싶은 학생
  • 자기주도형 프로젝트 경험을 쌓고 싶은 학생

학습 목표

  • 공공데이터 수집·정리·분석 경험
  • 그래프와 시각화를 활용한 결과 해석
  • 알고리즘 문제 해결 전략 설계
  • 발표 자료와 포트폴리오 완성

구성 요소

데이터 분석 및 시각화

실제 데이터를 정리하고 그래프를 만들어 분석 결과를 설명합니다.

알고리즘·문제 해결

고도화된 PBL 방식으로 문제를 정의하고 효율적인 해결 전략을 설계합니다.

포트폴리오 제작

프로젝트 결과를 보고서, 발표자료, GitHub 또는 Notion 포트폴리오 형태로 정리합니다.
