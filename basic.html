Basic 과정

기초 읽기와 반복 학습으로 코드 이해력을 세우는 과정

프로그래밍 입문 학생을 위한 온라인 보완학습입니다. 코드를 먼저 읽고, 따라 실행하고, 빈칸을 채우며 기본 구조를 익힙니다.

클래스 구성 보기전체 과정

대상 학생

  • 학교에 관련 과목이나 전담 지도가 부족한 학생
  • 프로그래밍을 처음 시작하는 중·고등학생
  • 혼자 강의를 들어도 코드 흐름이 잘 잡히지 않는 학생
  • 반복 실습과 쉬운 피드백이 필요한 학생

학습 목표

  • 코드 읽기 습관 형성
  • 변수·자료형·조건문·반복문의 기본 구조 이해
  • 실행 결과 예측과 오류 수정 능력 확보
  • 짧은 미션 완성으로 자신감 형성

구성 요소

입문: 코드 이해

라이브 코딩과 코드 읽기 중심으로 프로그램이 위에서 아래로 실행되는 흐름을 파악합니다.

기초 문법 반복

빈칸 채우기 방식으로 변수, 자료형, 입력, 출력, 연산 개념을 반복 학습합니다.

제어 구조 및 데이터

조건문·반복문 실행 흐름을 분석하고 리스트와 다차원 데이터 구조를 간단한 예제로 다룹니다.
