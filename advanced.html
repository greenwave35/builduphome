Advanced 과정

문제 해결과 데이터 응용으로 실전 감각을 키우는 과정

기초 문법을 배운 학생이 실제 문제를 분석하고, 코드를 수정·확장하며, 데이터 분석 미니 프로젝트까지 수행하는 단계입니다.

클래스 구성 보기전체 과정

대상 학생

  • 기초 문법은 알지만 응용 문제가 어려운 학생
  • 학교 과제나 대회 준비를 시작하려는 학생
  • 코드 수정, 디버깅, 흐름 추적 훈련이 필요한 학생
  • 데이터 분석 프로젝트를 작게 경험하고 싶은 학생

학습 목표

  • 자료형과 데이터 구조를 문제 해결에 활용
  • 코드 추적과 PRIMM 방식으로 코드 이해력 강화
  • 클래스·인스턴스 기반 설계 경험
  • 미니 데이터 분석 결과물 제작

구성 요소

자료형·데이터 응용

예제 기반 학습과 PBL로 리스트, 딕셔너리, 파일 데이터 활용 문제를 해결합니다.

제어 구조 심화

코드 추적, PRIMM, 코드 수정 과정을 통해 조건·반복·함수 구조를 확장합니다.

객체 활용 및 프로젝트

클래스와 인스턴스를 활용한 설계를 경험하고 데이터 분석 미니 프로젝트를 수행합니다.
