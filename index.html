학교 보완형 SW·AI 프로젝트 학습

학교 수업만으로 부족한 SW·AI 실전 경험을 채웁니다.

BuildUp은 심화선택 과목 보완, 수준별 프로그래밍 학습, 학생주도형 프로젝트, 포트폴리오 축적을 연결하는 온라인 학습 프로그램입니다.

과정 보기 필요한 학생 확인
Basic코드 읽기와 반복 실습으로 기초 이해력 형성
Advanced문제 해결·데이터 응용·미니 프로젝트 수행

누구에게 필요한가

지역과 학교 여건에 따라 SW·AI 학습 기회는 다르게 나타납니다. BuildUp은 다음 두 유형의 학생에게 특히 필요합니다.

실전 경험 보완형

많은 기회 속, 실제 경험은 부족한 학생

  • 학교 내 SW·AI 과목은 있으나 신청 경쟁이 치열함
  • 학생 수 대비 실습 장비와 프로젝트 기회가 부족함
  • 학원·온라인 강의 의존도는 높지만 체계적 결과물이 부족함
“배울 기회는 많다는데, 막상 해보려면 자리가 없어요.”
학습 기회 보완형

배우고 싶어도 배움의 기회 자체가 적은 학생

  • 학교에 SW·AI 관련 과목이 충분히 개설되지 않음
  • 전담 교사, 실습 환경, 피드백 기회가 부족함
  • 혼자 공부하다 막히면 해결하기 어렵고 동기 유지가 어려움
“어디서 배워야 할지 모르겠고, 학교엔 배울 수 있는 게 없어요.”

수준별 커리큘럼

기초 이해 → 문제 해결 → 프로젝트·포트폴리오로 이어지는 3단계 구조입니다.

01 Basic

  • 라이브 코딩
  • 코드 읽기
  • 빈칸 채우기
  • 조건문·반복문 흐름 분석
Basic 페이지로 이동
02 Advanced

문제 해결 과정

  • 자료형·데이터 응용
  • 코드 추적·PRIMM
  • 클래스·인스턴스 설계
  • 데이터 분석 미니 프로젝트

03 Expert

  • 고도화된 PBL
  • 알고리즘 문제 해결
  • 결과 발표 및 포트폴리오 완성
